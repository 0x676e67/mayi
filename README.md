# ant
### 主要技术栈
- 开发框架：Spring Boot 2.3.0.RELEASE
- 微服务框架：Spring Cloud 2020
- 安全框架：Spring Security + Spring OAuth 2.0
- 任务调度：Quartz
- 持久层框架：MyBatis Plus
- 数据库连接池：Druid、hikari
- 服务注册与发现: Nacos
- 客户端负载均衡：Ribbon
- 熔断组件：Sentinel
- 服务监控：Spring Boot Actuator、Spring boot Admin
- 网关组件：Spring Cloud Gateway
- 运行容器：Tomcat9.0
